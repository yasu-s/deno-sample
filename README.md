# 概要

- [Deno](https://deno.land/) と Visual Studio Code の拡張機能「[Remote - Containers](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)」を使用したサンプルです。

# 開発環境について

- Visual Studio Code 
  - Remote - Containers
- Docker Desktop

# 実行
```bash
deno run --allow-net main.ts
```
